Setup Instructions

  1. Open myaccount.google.com/security

  3. Go to App Passwords
  4. Create a new App Password for Mail / Windows Computer
  5. Copy the 16-character password
  6. Open .env in the project root and set:

  7. Restart the server: python manage.py runserver
