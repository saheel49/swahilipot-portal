Admin: Set up Event ID tracking in Google Forms

  1. Open your Google Form → add a Short answer question titled "Event ID" and mark it Required.
  2. Click ⋮ → Get pre-filled link → type any value in the Event ID field → Get link.
  3. In the generated URL find the part that looks like ?entry.XXXXXXXXX= — copy the entry.XXXXXXXXX key.
  4. Open events/models.py and set:
    GOOGLE_FORM_EVENT_ID_FIELD = "entry.XXXXXXXXX"
  5. Optionally add a second "Event Name" field and set GOOGLE_FORM_EVENT_NAME_FIELD the same way.
  6. From now on every form opened from this portal will auto-fill the Event ID — both from QR scans and direct registrations.
